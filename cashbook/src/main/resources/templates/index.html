가계부관리
일별 가계부관리
- 각 월별 가계부 관리로 가는 링크
- 전날, 다음날 가계부로 가는 링크
- 일별 합계 출력
- 입력, 수정(카테고리 직접입력 추가)
- 삭제
월별 가계부관리
- 각 월별 가계부 수입/지출 달력 형태로 출력
- 전달, 다음달 월별 가계부로 가는 링크
- 월별 비교 링크(년도별)
년도별 가계부 관리
- 작년, 내년 가계부로 가는 링크
- 각 월별 수입/지출 총합 출력
- 이번년도 수입/지출 총합 출력

게시판
- 10개 행, 5페이지씩 페이징
- 검색 추가
- 게시글 작성(이미지파일 업로드)
게시판 상세보기
- 이전글, 다음글, 게시글 목록 링크
- 게시글 상세정보출력
- 수정, 삭제

댓글
- 댓글 입력, 삭제
- 댓글 수정(jquery로 input태그 생성 후 ajax로 전송 성공하면 reload)
- 댓글 5개 행, 5페이지씩 페이징
